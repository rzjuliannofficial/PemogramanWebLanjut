## Requirements

- PHP >= 8.2
- Composer
- Laravel 12.x

## Instalasi

```bash
# Clone repository
git clone <repository-url>

# Install dependencies
composer install

# Copy environment file
cp .env.example .env

# Generate application key
php artisan key:generate

# Run development server
php artisan serve
```
